# Project Vision

## Name
ToDo API

## Objective
Create a REST API to manage my personal tasks.

The app will allow:

- Create tasks
- View the tasks list
- Update tasks
- Delete tasks
- Search tasks

## Public objective

Developers who need a small backend  to learn modern .NET architecture.

## Stack

- .NET 10
- FastEndpoints
- Dapper
- SQL Server
- FluentValidation
- xUnit
- Microsoft SQL
